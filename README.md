# guess-number-game


A simple and interactive number guessing game built using HTML, CSS, and JavaScript.

The goal is to guess the correct number between 1 and 100. The game gives hints after each attempt, making it fun and engaging.

##  Features

- Random number generated between 1 and 100
- User input for guessing
- Instant feedback:
  - Too high 📉
  - Too low 📈
  - Correct 🎉
- Tracks number of attempts
- Restart button to play again
- Clean and responsive UI


## 🛠️ Tech Used

- HTML
- CSS
- JavaScript 

## 🎮 How to Play

1. Open the game in your browser
2. Enter a number between 1 and 100
3. Click on **Check**
4. Read the hint:
   - "Too high" → try a smaller number  
   - "Too low" → try a bigger number  
5. Keep guessing until you get it right
6. Use the **Restart** button to play again
